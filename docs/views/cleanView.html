The clean view is the main view of the Clean application mode. This view offers a flexible interface to check plugins for errors, view the errors that are found, and then resolve them.

Error caching

zClean uses caching to avoid repeat work. See Error Caching Settings for more information.

Checking plugins for errors

  1. Make sure that the plugins you want to check for errors have the green Process badge displayed, and plugins that you don't want to process have the red Skip badge displayed. You can toggle between the two by clicking the badge.
  2. Click the big "Check for Errors" button at the bottom of the page.
  3. You can view errors and select resolutions for plugins while the error check is in progress.
  4. Once the error check is complete and you're ready to apply your resolutions, you can close the program. The Save Modal will be displayed automatically, from which you can select which plugins you want to save cleaned versions of.

Overview section

Before the error check is performed this section specifies how many plugins are queued to be checked for errors.

While an error check is being performed this section will specify the error checking progress and how many errors have been found.

Once the error check has completed this section will display a breakdown of the different types of errors that were found, with a short explanations for each type of error.

Plugin items

Before the error check is performed each plugin item will display the plugin's filename and a button that says "Process" or "Skip". Errors are displayed for plugins which have had cached errors loaded from disk.

While an error check is being performed, the plugin currently being checked will display a spinner. Plugins that are queued to be checked will display "queued" in parenthesis after their filename.

When an error check completes for a plugin, its errors will be displayed in groups.

Error groups

Errors are displayed in plugin items in groups, where each group corresponds to a type of error which is found in the plugin. You can expand groups to view errors and resolutions by clicking on them.

You can assign resolutions to all errors in a group from the dropdown by the group title. Selecting the Automatic option will have the program select the default preferred resolution for each error in the group. Selecting Manual will open the Resolve Modal for all the errors in the group. Selecting Ignore will set the resolution of every error in the group to ignore.

In addition to batch resolution, you can click on an individual error's resolution to set it from the resolve modal.
