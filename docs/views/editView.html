The edit view is the main view of the Edit application mode. This view offers a flexible interface to view and edit plugin files

Panes and tabs

The edit view currently offers two panes, which each allow you to have any number of tabs. In the future you will be able to split panes arbitrarily, and move tabs between panes.

You can create a tab by clicking the + button. You can close a tab by clicking the X button.
