The referenced by view allows you to view a list of records a record is referenced by.

Building references

The referenced by view won't display anything until you build reference information for plugins. See the Build References action in the context menu section on the tree view documentation page.

Linking

The referenced by view links to the first available record view it can find when it is created. When linked to a record view it will display referenced by information for the record open in the record view.

You can unlink a referenced by view from the record view it is linked to by clicking the button in the record view's address bar. When unlinked the only way to view a record on the referenced by view is through drag and drop from the tree view.

Columns

The referenced by displays the same columns as xEdit by default: Record, Signature, and File. You can edit columns by clicking the icon to open the edit columns modal.

Sorting

All columns, including custom columns, support sorting on the referenced by view. You can click on a column's header to sort by it or reverse its sort direction.

Hotkeys

  • Double Click - Opens the record in the referenced by view and the linked record view.
  • Enter - Opens the selected record node in the referenced by view and the linked record view.

Context menu

You can right-click a node in the referenced by view to open a context menu with actions. The context menu actions available are listed below:

Item Availability Hotkey Description
Open Enter Opens the selected node in the referenced by view and the linked record view.
