The record view allows you to view and edit values in records.

Record selection

You can open a record in the record view from the tree view or by entering a path into the address bar.

Element nodes

The record view is made up of a hierarchy of nodes which represent individual elements of a record. Nodes can be expanded or collapsed by clicking on the + / - button, and you can select a node by clicking on it. You can expand all nodes by clicking the + button in the Element Names header, and you can collapse all nodes by clicking the - button. See the hotkeys section below for additional interactions.

Columns

The first column on the record view contains element names corresponding to each node. The second column corresponmds to the original "master" version of the record. Subsequent columns correspond to overrides of the record.

Hotkeys

  • Double Click - Toggles the expansion of the clicked node and opens the edit value modal to edit it if it is an editable value.
  • Enter - Opens the selected node in the edit value modal if it is an editable value.
  • Right Arrow - Expands the last selected node if it is not expanded, else selects the first child node.
  • Left Arrow - Collapses the last selected node if it is expanded, else selects its parent node.
  • Up Arrow - Moves the selection up one node.
  • Down Arrow - Moves the selection down one node.
  • Page Down - Scrolls down one page of nodes.
  • Page Up - Scrolls up one page of nodes.
  • Delete - Deletes the selected nodes if possible.

Address bar

The address bar on the record view can be used to navigate to a specific record via a path, or to navigate through records that were previously viewed. You can close the address bar by clicking the X button on it. You can open or focus the address bar by pressing Ctrl + A.

Path formats

Below are examples of some of the most commonly used valid path formats:

  • FileName.esp\01234567 - Hexadecimal Form ID path.
  • FileName.esp\RecordEditorID - Editor ID path.
  • FileName.esp\"FULL Name" - FULL Name path.

Hotkeys

Note: all hotkeys only work when the address input is focused unless otherwise noted.

  • Enter - Navigates to the entered path.
  • Backspace OR Alt + Left Arrow - Navigates back to the previously viewed record. Works when the address bar is not focused.
  • Alt + Right Arrow - Navigates forward to the next viewed record. Works when the address bar is not focused.

Context menu

You can right-click anywhere in the record view to open a context menu with actions. The context menu actions available are listed below:

Item Availability Hotkey Description
Add Visible when right-clicking an unassigned element or array element in an editable record. Insert Adds the element to the record. Adding an element to an array will append a new element to it.
Delete Visible when right-clicking a removable element in an editable record. Delete Removes the element from the record.
Edit Visible when right-clicking a value element in an editable record. Enter Opens the selected node in the record view.
Copy Ctrl + C Unimplemented.
Copy Path Ctrl + Shift + C Unimplemented.
Paste Ctrl + V Unimplemented.
