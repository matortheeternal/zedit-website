zEdit supports the following arguments:

Argument Description
-appMode="AppName" Sets the application mode to AppName. Supported application mode names are: edit, merge, and clean.
-profile="ProfileName" Sets the current profile to ProfileName and starts the application with that profile.
-dev Opens the developer console before the main application window gets created. Using this argument is recommended for debugging modules.
-skipLoadOrder Automatically loads your current load order when the application mode is set to edit or clean.
-verbose Enables verbose logging.
--disable-transparency Disables transparency for the progress window. Use this if you have window transparency disabled or if your OS does not support window transparency.
--inspector-fix Makes the progress window non-modal, allowing you to interact with the main application window and the inspector when it is embedded in the main application window. This argument is deprecated and may be removed in a future release. (detach the inspector to a separate window instead)
