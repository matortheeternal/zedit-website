The modal API allows you to open a modal with an arbitrary HTML template and controller at any point in the application's lifecycle via an angular event.

It's always best to $emit events from a child scope. If you don't have a child scope to emit your event from you can $broadcast it from the $rootScope instead.

Unfocus Modal Directive

The unfocus modal directive provides an easy way to allow users to close your modal by clicking outside of it. To use it, simply put unfocus-modal in your modal-container div. You can tell the unfocus modal directive to execute a function before your modal closes via unfocus-modal="closeModal()", where closeModal is a function in your modal's controller.
