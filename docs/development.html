Technologies

zEdit uses the following technologies:

  • ES6 JavaScript - Programming Language
  • NodeJS - JavaScript Runtime
  • Electron - Platform
  • AngularJS 1.5 - JavaScript Framework
  • SASS - CSS Extension Language
  • CSS3 - Stylesheet Language
  • HTML5 - Markup Language

APIs

zEdit offers a number of APIs for developers to use.
