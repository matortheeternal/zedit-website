zEdit offers several application modes other than the default "Edit" mode. Different application modes are used to accomplish different tasks. Application modes can be selected from the Start View.

Currently planned/implemented application modes include:

Application Mode Description
Edit This application mode offers functionality for viewing, editing, and saving plugin files. See Edit View for more information.
Clean Application mode which offers plugin cleaning functionality. See Clean View for more information.
[Planned] Merge Planned application mode which will offer functionality similar to Merge Plugins Standalone.
[Planned] Smash Planned application mode which will offer functionality similar to Mator Smash.
