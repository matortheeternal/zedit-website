The load order modal allows you to select an order of plugins to load. It appears automatically when you press the Start Session button on the Start View.

The load order modal cannot be closed because opening it requires setting the game mode and initializing definitions in XEditLib.

Plugin items

The load order modal lists a number of plugin items. Each item can be checked or unchecked to indicate whether or not it should be loaded, and displays both the target load order (in hexadecimal) and filename of the plugin to load. A maximum of 255 plugins can be loaded due to limits of the hexadecimal FormID system.

Drag and drop

You can drag and drop plugins to change their order. This is a backup mechanism for situations in which your load order was not parsed correctly, or when you want to make a small adjustment without restarting the program.

Plugin states

  • Disabled - Plugins are colored red when one or more of their masters are unavailable to load. Plugins are considered unavailable to load if they are not found in your data directory or if they are disabled. You can move your mouse over a disabled plugin to get a tooltip specifying the masters which are unavailable to it. Disabled plugins cannot be activated.
  • Required - Plugins are colored green when an active plugin requires them as a master. You can move your mouse over a required plugin to get a tooltip specifying the active plugins which require it.

What you see if what you get

When a plugin is activated any masters it requires will automatically be activated as well. Similarly, when a plugin is deactivated, any plugins which require it will be deactivated as well. This means that the load order modal presents the exact load order that will be loaded into zEdit - what you see is what you get.

Hotkeys

  • Ctrl + Click - Toggles the selection of the plugin clicked, preserving the current selection.
  • Shift + Click - Selects all plugins between the clicked plugin and the previously selected plugin.
  • Ctrl + Shift + Click - Same as Shift + Click, but also preserves the current selection.
  • Up Arrow - Moves the selection up one plugin. Can be used for multi-selection if Shift is held.
  • Down Arrow - Moves the selection down one plugin. Can be used for multi-selection if Shift is held.
  • Space - Toggles the activation of the selected plugins.
  • Ctrl + Space - Activates the selected plugins.
  • Shift + Space - Deactivates the selected plugins.
  • Ctrl + A - Selects all plugins.
  • Escape - Clears your current plugin selection.
  • Enter - Loads the active plugins. Equivalent to clicking the OK button.
