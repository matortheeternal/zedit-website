The help modal allows you to view help topics and documentation for zEdit. You can open the modal:

  • Pressing Ctrl + Shift + H.
  • By clicking the ? icon in the title bar.

Navigation

The left section of the Help Modal displays a tree of help topics. Every node in the tree corresponds to a help page which can be viewed, though some pages may only display lists of child topics.

You can navigate back/forward relative to a topic through the arrow buttons at the header of each page.
