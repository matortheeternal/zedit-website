The manage extensions modal allows you to manage installed modules and themes. You can open this modal by:

  • Pressing Ctrl + Shift + E.
  • Clicking the icon in the title bar.

Installed modules tab

The installed modules tab lists your currently installed modules. Each module item lists the module's name, version, author, and description. There's also the Uninstall button which uninstalls the module, deleting it from disk.

You can click the Install Module button to install a module from an archive or module.json file. Note that installing or uninstalling modules does not activate them in the program immediately - a restart is usually necessary.

Installed themes tab

The installed themes tab lists your currently installed application themes. Each theme item lists the theme's name, version, author, and description. There's also the Apply button which applies the theme, and the Uninstall button which uninstalls the theme, deleting it from disk.

You can click the Install Theme button to install a theme from an archive or .css file.
