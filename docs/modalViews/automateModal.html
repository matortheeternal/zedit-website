The automate modal can be used to execute user scripts.

You can open the automate modal by:

  1. Right-clicking in the Tree View and clicking the Automate option.
  2. Pressing Ctrl + M while the tree view is focused.

Selecting a script

You can change the selected script by clicking the script dropdown Clicking the script's name directly will edit the name.

Running a script

Click the OK button or press Ctrl + Enter to run the currently selected script.

Creating a new script

You can create a new script by clicking the New button or pressing Ctrl + N.

Saving a script

You can save changes made to a script at any time by clicking the Save button or by pressing Ctrl + S. Changes will be automatically saved when the modal is closed or the script is executed.

Renaming a script

You can rename a script by clicking on its name or by pressing F2 while it is selected. The script will be moved to the new filename when it is next saved.

Deleting a script

You can delete the currently selected script by clicking the Delete button or by pressing Delete.
