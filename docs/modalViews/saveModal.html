The save modal allows you to specify plugins to save, and displays the while saving plugins or closing the application. You can open the save modal by:

  • Pressing Ctrl + S on the Edit View.
  • Attempting to close the application when there are plugins that have been modified.

Clean Mode

When using the application in clean mode, the save modal will close the program even if it is opened with Ctrl + S. This will be fixed at a future point in time.

Hotkeys

The plugins list on the save modal responds to the same hotkeys as the Load Order Modal

Actions

  • Click the Save button to save the checked plugins.
  • Click the Discard button to discard changes to the plugins. Note: This button is only visible when you're closing the application.
  • Click the Cancel button to not save any changes and keep the application open.
