The tree view settings tab allows you to adjust the behavior and appearance of the Tree View.

Setting Description
Show group signatures Enables the display of signatures on record groups. E.g. with this enabled, the Armor group will be displayed as ARMO - Armor.
Prompt on deletion If enabled a prompt will be shown to confirm the deletion of any records or groups on the tree view.
Show separate file headers If disabled file headers will not be displayed in the tree view.
If enabled file headers will be displayed as a separate node in the tree.
