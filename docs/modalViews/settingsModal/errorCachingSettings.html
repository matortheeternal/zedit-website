zClean caches errors based on plugin filenames and MD5 hashes. Error cache files are saved in minified JSON in the cache folder. The error caching settings tab allows you to view and manage error cache files.

Disabling error caching

You can disable error caching by unchecking the Cache errors checkbox. Error caching is enabled by default, and is recommended.

Clearing the cache

You can clear the entire error cache by clicking the Clear button.

You can delete all cache files for a particular plugin by clicking the Delete button by it.

You can delete an individual cache file by expanding the cache files section and clicking the X button by a cache file entry.
