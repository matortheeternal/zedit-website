The archive creation settings tab allows you to configure how Bethesda archive files are created when merging plugins.

Setting Description
Files to include A list of glob expressions used to find files to include when building a Bethesda archive file. You can add a new expression by clicking the "Add" button and delete an existing expression by clicking the red "X" button by it.
Maximum archive size The maximum filesize an archive is allowed to reach before files are moved into a different archive or left loose.
Minimum files per archive The minimum number of files required to create a Bethesda archive file. If there are less than this many files available to be archived, a Bethesda archive file will not be created.
Create textures archive If enabled, a ` - Textures` archive will be created if the maximum archive file size is reached on the first archive.
Creature multiple archives If enabled, an extra archive will be created whenever the maximum archive file size is reached on an archive. Extra archives are named in the format: <BaseName> - <ArchiveNumber>.<ArchiveExtension>.
