The mod manager integration allows zMerge to integrate with your mod manager for the purpose of detecting assets, determining asset priority, and more.

Setting Description
Mod manager The mod manager you want to integrate with. Supported mod managers are Mod Organizer 2, Mod Organizer, Vortex, and Nexus Mod Manager.
MO2 Instance This is the name of the MO2 instance you want to integrate with. This setting is only available for when mod manager is set to Mod Organizer 2.
Mod manager path The path to the folder where the mod manager executable is located.
Mod manager mods path The path to the folder where the mod manager mod folders are located.
