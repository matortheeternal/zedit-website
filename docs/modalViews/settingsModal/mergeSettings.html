The merge settings tab allows you to configure the merge output path and merge integrations.

Setting Description
Merge output path The path to the folder where you want zMerge to output merges and relinked scripts. Separate folders will be created at the merge output path for each merge built.
Disable plugins If checked, merged plugins will be automatically disabled in plugins.txt once a merge is successfully built.
Disable mods If checked, mods will be disabled in your mod manager once a merge with copy general assets enabled is successfully built. This integration is only available when using Mod Organizer or Mod Organizer 2 currently.
