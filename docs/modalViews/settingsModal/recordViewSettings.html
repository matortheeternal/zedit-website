The record view settings tab allows you to adjust the behavior and appearance of the Record View.

Setting Description
Auto-expand nodes When enabled, all record nodes will be expanded by default when you open a record in the record view.
Show array element indexes If enabled, array indexes will be appended to the name of elements in arrays. Indexes are always displayed for unsorted arrays. Indexes are only displayed for sorted arrays when a record column has been focused.
Prompt on deletion If enabled a prompt will be shown to confirm the deletion of any elements on the record view.
Default column widths These values correspond to the default column widths (measured in pixels) used on the record view when a record is opened.
