The profiles modal allows you to manage and create profiles. You can open it by:

  • Clicking the button on the Start View.
  • Clicking the button in the title bar while on the Start View.
  • Pressing Ctrl + Shift + S while on the Start View.

Profile Fields

Field Description
Name The name of the profile. This is what is displayed in the dropdown on the Start View.
Game The game you want to use the profile for. Can be any supported game mode.
Path The path to your game installation folder. You can click the folder icon to browse for it.
Languge Language mode to use. Currently this only applies to the localization used when loading game files.

Default Profile

You can change the profile will that is selected when the application starts by selecting a profile from the default profile dropdown.
