The resolve modal allows you view error records and to select resolutions. You can open the resolve modal by:

  • Selecting the Manual group resolution for an error group on the Clean View.
  • Clicking on an individual error item's resolution on the Clean View.

Navigation

Selecting an error resolution will navigate to the next error. You can also click the Next button to navigate to the next error. You can click the Previous button to navigate to the previous error. When you've reached the last error, the Done button will appear instead of the next button.

Error resolutions

The default error resolution applied to an error is the first resolution for the error type available for the record the error is in.

Hotkeys

  • W, Right Arrow - Navigates to the next error.
  • Q, Left Arrow - Navigates to the previous error.
  • 1, 2, 3, 4... - Selects the corresponding resolution.
  • Escape - Closes the modal.
